Hello all,

I'm proud to announce an immediate availablity of supporting modules for
TableLayout and FormLayout layout managers.
NetBeans 3.6, 4.0, and 4.1 are supported.

Modules available for download from
https://developer.berlios.de/projects/nblayoutpack/

Changes:
- bug fixes
- new icons for Palette
- TableLayout library updated to version 3.4
- JavaDoc documentation for TableLayout added
